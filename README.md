# Bari-MarketPlace
Bari-MarketPlace is a web platform that integrates a marketplace, and weight loss calculator for treatments and products, connecting patients directly with doctors.


## Technologies

### Backend
- Nest.js
- PostgreSQL
- Prisma

### Frontend
- Next.js
-  Typescript
